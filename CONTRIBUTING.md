# Contributing to Vidilearn

Thank you for your interest in Vidilearn! We welcome contributions to help make this tool better for the AI community.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Alfo-Tech-Lab/vidilearn.git
    cd vidilearn
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run tests:**
    ```bash
    npm test
    ```

## Development Workflow

- **Branching:** Create a new branch for your feature or bugfix: `git checkout -b feature/my-new-feature`.
- **Coding Standards:** We follow standard JavaScript ESM conventions.
- **Testing:** Ensure all tests pass before submitting a PR. Add new tests for new features.
- **Commit Messages:** Use descriptive commit messages.

## PR Checklist

- [ ] Tests pass (`npm test`).
- [ ] Documentation updated (if applicable).
- [ ] New features include tests.
- [ ] Commit messages follow conventions.

## Feature Requests & Bug Reports

Please open an issue on the GitHub repository to discuss major changes before submitting a PR.
