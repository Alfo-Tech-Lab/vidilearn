# Product Roadmap

This document outlines the milestones for expanding Vidilearn into a local AI knowledge infrastructure platform.

---

## Phase 1: Powerful Core Engine (Completed)
- [x] Universal Input Support (PDF, DOCX, EPUB, RSS, Git, Audio local transcription).
- [x] Local Vector Memory (Embedded SQLite).
- [x] Local Embeddings (Transformers.js offline).
- [x] Hybrid Search Engine (FTS5 + Cosine Similarity).
- [x] Auto Knowledge Graph (Mermaid/Obsidian exports).

## Phase 2: Agentic Features (Completed)
- [x] Deep Research Mode (Multi-source YouTube synthesis).
- [x] Autonomous Watcher (Delta monitoring).
- [x] MCP Super Mode (Schema RPC server).
- [x] AI Pipeline Templates (RAG presets).

## Phase 3: Viral & Diagnostic Features (Completed)
- [x] Highlights pacing clip extractor.
- [x] Cornell study notes and Anki flashcard exporter.
- [x] Fine-tuning datasets compiler.
- [x] Interactive terminal dashboard (TUI).

## Phase 4: Scaling & Integrations (Upcoming)
- [ ] Native integration with local vector databases (LanceDB, Qdrant).
- [ ] Browser Companion extension to clip text/videos in 1-click.
- [ ] Offline audio diarization (determining speaker switches).
